The Proposal

Where Math
Meets the Machine

While Python dominates data science headlines, .NET developers have a quiet, powerful secret: F#. This intensive two-day workshop bridges the chasm between academic mathematics and production-grade engineering.

Attendees master translating mathematical specifications into elegant, type-safe F# code — covering linear algebra, statistics, probability, calculus, symbolic differentiation, and numerical simulation. Every concept is grounded in real case studies: soccer analytics, housing price models, bungee physics, predator–prey ecosystems, and credit scoring.

The Book Behind the Workshop

Applied Mathematics
with F#

This workshop is drawn directly from the Springer publication by Sudipta Mukherjee (November 2025). The book highlights the enormous potential of F# in scientific computing — presenting solutions to diverse real-life problems through applied mathematics and functional programming.

It covers F# collections, vectors, matrices, descriptive statistics, set theory, probability, and calculus — using Plotly.NET for interactive visualisation, AngouriMath for symbolic differentiation, and Math.NET for numerical computation.
